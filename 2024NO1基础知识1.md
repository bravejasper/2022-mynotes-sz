##### 9.git使用命令

```
1.git init
2.git add . （".“代表该目录下的所有文件，如果不想上传所有文件，把”."换成想要上传的文件名）
3.git commit -m 'first'（first可任意换，相当于描述）
4.git remote add origin https://github.com/xxx/vvv.git
5.git push -u origin master
```
